# Angry Birds Style Slingshot Prototype (WhiteHat Jr Project)

This project is an early **physics-based slingshot experiment** inspired by Angry Birds.  
It was created during my early programming learning days with WhiteHat Jr and demonstrates:

  - Slingshot mechanics  
  - Physics-style object launching  
  - Collision with blocks and pigs  
  - Background sprites and simple animation  

This is **not a full game** ‚Äî it is a **mechanics prototype** focused on slingshot + collision logic.
